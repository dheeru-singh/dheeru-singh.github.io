Cinema Snapshot

Project overview

Cinema Snapshot is a movie search application. you can search any movie by their title. Their is list of popular movies on the popularmovies page. and also the list of top rated movies list. on clicking on detail button you can find the detail of movies.

Technologies Utilized

  • NodeJS
  • JavaScript
  • Express
  • Bootstrap
  • MVC
  • Handlbars
  • OMDB API and The Movie DB API
